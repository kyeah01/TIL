콜백함수는 주로 비동기 처리 모델에서 사용된다.

동기(직렬) / 비동기(병렬)

동기(직렬)은 - 위에서부터 하나씩 로딩되는 것

비동기(병렬)은 - 동시다발적으로 로딩되는 것.

브라우저는 스레드가 하나

ie 7 는 탭 모두가 하나의 스레드로 연결되어있었고, 하나의 탭이 죽으면 모든 탭이 로딩되지 않음.(동기방식)

스레드는 하나이지만, 동기가 아니라 비동기방식(병렬) 으로 처리하게끔으로 바뀜
