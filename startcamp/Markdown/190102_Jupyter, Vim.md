### git ignore  파일 설정

~~~bash
touch .gitignore
ls -al
echo "test.txt" >> .gitignore
cat .gitignore

~~~

vim  켜서 ignore 파일에 한번에 넣기(echo사용 안하고!)

~~~bash
vi .gitignore
~~~

command 와 insert는 `i`와 `esc`로 이동
